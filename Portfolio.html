Lab 1 - Temperature Sensor

We designed a thermometer with web functionality. Our thermometer system is made using an enclosed box with a circuit within. An Arduino UNO is used to provide power to the circuit inside the box, as well as handle I/O logic to the circuit components. The Arduino is connected to an on-off switch that, when flipped, turns the device on and off. Connected to the power switch is a 5V battery that can supply power to the device in the case that the power cord to the Arduino is not plugged in. The Arduino is connected to four main components: a Raspberry Pi Pico W, the 4-digit 7-segment LED display, a pushbutton, and the DHT22 Temperature sensor. A level shifter is used to convert a 5V power source into a 3.3V power source for the Raspberry Pi Pico W. The Pico handles wireless communication from the physical circuit to the computer, which then can send text messages to a cell phone. The display and pushbutton are enclosed inside the box, and when the button is pressed, the display will show the current temperature in degrees C. The temperature sensor is attached to a 2.0 meter-long cable that is connected to the circuit inside the box. The sensor, the Arduino power cable, and the Raspberry Pi power cable can all be easily plugged and unplugged as needed.

Lab 2 - Electric Eye

The goal of this lab is to design an “electric eye” light beam-interrupter safety system. The system consists of a transmitter that projects a beam of light onto a receiver. When an object physically obstructs the beam, a warning light is powered on and turns off when the obstructing object is removed. The purpose of the lab is to explore how to process noisy signals, namely by designing a device that can read a beam of light in a variety of lighting conditions reliably. We were supplied the transmitter device schematic and parts and built it accordingly for testing purposes. The transmitter uses a 555 timer and a LED driver circuit to generate a modulated transmit beam, which the receiver then reads.

Lab 3 - Portfolio Website

This is the project you're looking at right now! The primary purpose of this lab is to get you a preliminary experience in establishing a website and to highlight the importance of well-structured and intuitive user interface (UI) and user experience (UX) design. Since the UI is typically the primary portal via which users interact with a system, the success or failure of a system design is often dictated by the nature of the user experience provided by the interface. Poor interface design may compromise otherwise soundly designed systems.
